# Tela_azul_PyAutoGui

Pequeno exercício usando PyAutoGui

Ao baixar e executar este código tenha em mente que:

1 - A biblioteca pyautogui deve estar instalada no ambiente de trabalho.
2 - O código deve ser executado no vscode ou IDEs que suportem o pyautogui.
3 - A página html deverá ser inicializada com o live server, caso contrário não será possível acessar o site.
4 - O código do bloco de notas será executado independente do live server.

Este código é seguro e tem propósito somente educacional e/ou lazer, nada será afetado na sua máquina.
